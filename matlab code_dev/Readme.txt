This is small guide that helps how you can run the project.
The main program that runs all the files of this project is Mainheader.m.
Please run Mainheader.m and then you will get the Menu to choose different options.
After observing the results, please go to prompt and Enter digit 1 if you like to continue the program to see more result, or just enter any digit 
if you do not like to continue any more.
